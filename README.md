# N-Body Gravitational Simulation

Интерактивный симулятор гравитационного взаимодействия N тел с 3D-визуализацией, системой сохранений и готовыми пресетами.

## Возможности

### Физика
- Гравитационное N-тел взаимодействие (закон Ньютона) с **субшаговым интегрированием** (100 шагов/кадр)
- **Softening** — сглаживание силы при сближении тел, предотвращает численную нестабильность
- **Обнаружение столкновений** — при пересечении радиусов тела сливаются с сохранением массы и импульса
- Защита от **туннелирования**: минимальное расстояние отслеживается внутри субшагов

### Визуализация
- **3D-график** с полными траекториями тел (история не обрезается)
- **2D-проекция** на выбранную плоскость (XY / XZ / YZ)
- **Режим размеров** — отображение тел пропорционально их радиусу (логарифмическая шкала)
- Таблица координат, скоростей и масс в реальном времени
- Автоматическое масштабирование области отображения

### Управление
- **Скорость симуляции** — слайдер от 1x до 20x (число физических шагов за кадр)
- Пауза / продолжение / сброс к начальным условиям
- Настройка временного шага (dt)

### Пресеты
Готовые наборы тел с физически корректными начальными условиями:

| Пресет | Тела | dt |
|---|---|---|
| Двойная звезда | 2 звезды по 2M☉ на круговой орбите | 1 сутки |
| Земля — Луна | Реальные массы и орбита | 1 час |
| Солнце — Земля — Юпитер | Реальные параметры | 1 сутки |
| Три тела (треугольник Лагранжа) | 3 равные звезды, устойчивое решение | 1 час |

### Данные
- Система **сохранений** в JSON — создание, загрузка, удаление конфигураций
- **Экспорт** истории симуляции в Excel (.xlsx), отдельный лист для каждого тела
- Поддержка ввода в научной нотации (`1.5e10`)

## Параметры тела

Каждое тело задаётся следующими полями:

| Параметр | Описание |
|---|---|
| Имя | Название тела |
| X, Y, Z | Начальные координаты (м) |
| Vx, Vy, Vz | Начальные компоненты скорости (м/с) |
| Масса | Масса тела (кг) |
| Радиус | Радиус тела (м) — используется для столкновений и режима размеров |

## Установка и запуск

**Требования:** Python 3.9+

```bash
pip install -r requirements.txt
python modelNtel.py
```

## Сборка Windows EXE

EXE-файл собирается автоматически через **GitHub Actions** при каждом push в `main`.

Скачать: вкладка `Actions` → последний запуск `Build Windows EXE` → раздел `Artifacts`.

Запустить сборку вручную: `Actions` → `Build Windows EXE` → `Run workflow`.

Локальная сборка:
```bash
pip install pyinstaller
pyinstaller N-Body-Simulation.spec --noconfirm
# EXE будет в папке dist/
```
